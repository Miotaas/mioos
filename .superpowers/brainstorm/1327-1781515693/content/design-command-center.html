Design notes
• The Executive Brief header is purely display — it reads from the existing API (/api/company/dashboard) + a new lastVisit timestamp stored in localStorage.
• Bullet content is auto-generated from real data: new opportunities created since lastVisit, approvals, completed assignments.
• "Biggest Opportunity / Risk / Recommended Action" already exists in the current CompanyCommandCenter data model — just surfaces it differently.
• Every grid section has a "View → [page]" link in its header to the corresponding drill-down page.
• The 6-section grid is the existing 3-column layout, enhanced with Workforce and Intelligence sections and styled "View →" links.
